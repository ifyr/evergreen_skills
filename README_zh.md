# Evergreen Skills for Software Engineers

> **基于 [Evergreen Skills for Software Developers](https://github.com/romenrg/evergreen-skills-developers) 改编，原作者 [Romén Rodríguez-Gil](https://www.romenrg.com/about/)。** MIT 许可证。

一份中英双语 Wiki，记录经得起时间考验的**软件工程"常青技能"**——软件开发最佳实践、跨框架通用原则和可迁移技能。

> 框架会过时，语法可查阅。真正衡量工程师价值的，是那些**无法轻易搜索到、需要长期积累**的工程素养。

## 目录结构

```
wiki/
├── en/    英文 Wiki（21 页）
└── zh/    中文 Wiki（21 页）
```

每份 Wiki 是自包含、互相关联的知识库：

| 章节 | 涵盖内容 |
|------|----------|
| **非技术技能** | 沟通能力、团队协作、开发流程、问题解决能力、心态 |
| **编程原则** | 控制流、布尔代数、OOP、SOLID、GRASP、函数式编程、声明式 vs 命令式 |
| **数据结构** | 缓存/记忆化、栈 vs 堆 |
| **整洁代码** | 命名、SemVer、代码注释、组合优于继承、TDD |
| **源代码管理** | 微提交、约定式提交、分支策略、依赖地狱 |
| **技术协作** | Code Review、结对编程 |
| **DevOps 实践** | CI/CD、构建自动化、测试金字塔、特性开关 |
| **语言理论与底层** | 编译 vs 解释、类型系统、正则、懒加载、性能剖析、并发 |
| **领域专项** | 前端、后端、架构、基础设施、安全 |

## 快速开始

从 Wiki 首页进入：

- [English Wiki](wiki/en/Home.md)
- [中文 Wiki](wiki/zh/Home.md)

每页均有前后导航链接，可按顺序阅读或跳转到任意主题。
