# Evergreen Skills for Software Engineers

> **Based on [Evergreen Skills for Software Developers](https://github.com/romenrg/evergreen-skills-developers) by [Romén Rodríguez-Gil](https://www.romenrg.com/about/).** Licensed under MIT.

A bilingual (English / Chinese) wiki documenting **"evergreen skills"** — software development best practices, cross-framework principles, and portable skills that stand the test of time.

> Unlike framework-specific knowledge that quickly becomes outdated, the skills in this wiki focus on **what cannot be easily googled** and **takes years to master**.

## Structure

```
wiki/
├── (English wiki, 21 pages)
└── zh/   (Chinese wiki, 21 pages)
```

Each wiki is a self-contained interlinked knowledge base:

| Chapter | Topics |
|---------|--------|
| **Non-Technical Skills** | Communication, Teamwork, Development Process, Problem-Solving, Mindset |
| **Programming Principles** | Control Flow, Boolean Algebra, OOP, SOLID, GRASP, FP, Declarative vs. Imperative |
| **Data Structures** | Caching / Memoization, Stack vs. Heap |
| **Clean Code** | Naming, SemVer, Code Comments, Composition over Inheritance, TDD |
| **Source Code Management** | Micro Commits, Conventional Commits, Branching, Dependency Hell |
| **Technical Collaboration** | Code Review, Pair Programming |
| **DevOps Practices** | CI/CD, Build Automation, Test Pyramid, Feature Flags |
| **Language Theory** | Compiled vs. Interpreted, Type Systems, Regex, Lazy Loading, Profiling, Concurrency |
| **Field-Specific** | Front-End, Back-End, Architecture, Infrastructure, Security |

## Quick Start

Start from the wiki home page:

- [English Wiki](https://ifyr.github.io/evergreen_skills/)
- [中文 Wiki](https://ifyr.github.io/evergreen_skills/zh/)

Each page links to the previous and next, so you can read sequentially or jump to any topic.
