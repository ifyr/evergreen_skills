# Evergreen Skills for Software Engineers

**A bilingual wiki of software engineering skills that stand the test of time.**

> 一份中英双语 Wiki，记录经得起时间考验的软件工程常青技能。

Programming languages evolve, frameworks come and go, but certain skills remain timeless. This project documents those skills — from communication and critical thinking to SOLID principles and observability — in a structured, interlinked, bilingual format.

Every concept is explained in depth with original source material, organized as a self-contained 21-page wiki.

**Based on the original work by [Romén Rodríguez-Gil](https://github.com/romenrg/evergreen-skills-developers).**

---

编程语言不断演进，框架来了又走，但有些技能始终长青。本项目将这些技能——从沟通和批判性思维到 SOLID 原则和可观测性——以结构化、互相关联、中英双语的形式记录下来。

每个概念均有原文来源的深度解释，组织为自包含的 21 页 Wiki。

**基于 [Romén Rodríguez-Gil](https://github.com/romenrg/evergreen-skills-developers) 的原始项目改编。**
